This function finds dead and muted strips in the tracker, which one can safely assume to be equal to the set of strips that have never been hit if the data sample is big enough.

All variable are referenced from the maincode and most of those are inserted as inputs only to avoid allocating new memory inside this function.

Parameters
fileThe input file given by the User in the CreateTree.cpp code
NHitVariable given by reference from the maincode: used to read the file
cicloUtility variable used to keep count of the while iteration during the file reading
LayersIDContains the ID's of the layers of the tracker
mappingIt maps raw string info of the layers to the int ID's
MapDeiVettoriMutatiIt maps the layer ID's to the vectors containing the dead strips

It is used for reading the file, it is used in fact as a placeholder for each file line using the command getline()

Vector containing the TH1F pointers of the histograms of hits for the Layers: those histograms have 1536 bins which is equal to the number of strips in each layers

Maps Layer's ID to its histogram pointer

In this for loop we create the TH1F pointers and we fill the HitCounts vector and link layers' ID to its histogram.

In this for-loop we check for each layer the counts of hits for each strip, if the count is 0 then the strip is added to the list of dead/muted strips

Push back the strip ID into the vector corresponding to the layer's ID

Close file

Delete pointers
