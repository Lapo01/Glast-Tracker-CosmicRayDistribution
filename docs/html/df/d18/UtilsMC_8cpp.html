This function generates four random numbers: zenith angle theta, azimuth angle theta, point coordinates (x0, y0) on the plane of the top layers and then accordingly generates the projections.

Theta is generated using the Inverse Transforming Sample Method, the pdf is a distribution sin(theta)*cos^3(theta). Phi is generated uniformly and also the coordinates (x0, y0).

Parameters
auxiliaryuniformly generated variable used in the Inverse Transforming Sample Method
thetaZenith Angle, it is one of the four generated numbers.
phiAzimuth Angle, it is one of the four generated numbers.
x0X coordinate of point that belongs to the track on the plane of the top layer, it is one of the four generated numbers.
y0Y coordinate of point that belongs to the track on the plane of the top layer,it is one of the four generated numbers.
